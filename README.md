\# Bridge Rectifier Simulation



This is my first electronics simulation project for EEEN204.



\## Aim



The aim of this project is to simulate a bridge rectifier circuit and understand how AC voltage can be converted into DC voltage.



\## What is a bridge rectifier?



A bridge rectifier uses four diodes to convert both the positive and negative halves of an AC signal into a positive output signal.
